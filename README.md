## 스터디 규칙

- Programmers, LeetCode 풀면서 부딪혀보기
- 각각 1시간 푸는 시간 (총 3문제)
    - 1시간: 40분은 혼자 풀고, 나머지는 같이 토론

## PR 규칙

- 이 레포를 fork 후, 각자의 폴더를 만들어서 PR
    - PR만 하고 merge는 꼭 나중에! -> 코드리뷰 해야 함
    - 코드리뷰 다 끝나면 merge
- 폴더 경로: (site/name/week/file)
